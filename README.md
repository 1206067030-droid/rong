# YINGRONG INTL LIMITED 官網

這是一個基於 Next.js 14 + TypeScript + Tailwind CSS 的極簡風格企業官網，採用蘋果官網設計風格，提供完整的產品展示、最新消息、案例作品和聯絡功能。

## 技術栈

- **框架**: Next.js 14 (App Router)
- **語言**: TypeScript
- **樣式**: Tailwind CSS
- **圖標**: Lucide React
- **部署**: Static Export (支持靜態部署)

## 頁面結構

### 前台頁面
- **首頁** (`/`): 全幅 Banner、核心產品展示、品牌優勢、最新消息預覽
- **關於我們** (`/about`): 品牌介紹、核心優勢、經營理念、國際貿易優勢
- **產品中心** (`/products`): 產品分類篩選、卡片式展示、價格區間顯示
- **產品詳情** (`/products/[id]`): 多圖展示、產品參數、特色介紹、價格展示
- **最新消息** (`/news`): 文章列表、分類展示
- **文章詳情** (`/news/[id]`): 文章內容、上下篇切換
- **案例作品** (`/cases`): 案例圖集、分類篩選、彈窗預覽
- **聯絡我們** (`/contact`): 公司資訊、Google 地圖、線上諮詢表單
- **隱私權政策** (`/privacy`): 隱私權政策說明
- **404 頁面** (`/not-found`): 極簡風格錯誤頁面

### 後台管理系統
- **登入頁面** (`/admin/login`): 管理員登入
- **控制台** (`/admin/dashboard`): 數據統計、最新留言、快速操作
- **產品管理** (`/admin/products`): 產品列表、狀態切換、編輯/刪除
- **文章管理** (`/admin/articles`): 文章列表、狀態切換、編輯/刪除
- **案例管理** (`/admin/cases`): 案例列表、狀態切換、編輯/刪除
- **留言管理** (`/admin/messages`): 留言列表、標記已讀、刪除
- **網站設定** (`/admin/settings`): SEO 設定、功能開關

## 安裝和運行

### 1. 安裝依賴
```bash
npm install
```

### 2. 運行開發服務器
```bash
npm run dev
```

### 3. 打包生產版本
```bash
npm run build
```

### 4. 運行生產服務器
```bash
npm run start
```

## 部署說明

### 靜態部署
本項目支持靜態部署，打包後會生成 `out` 文件夾，可直接部署到任何靜態服務器：

```bash
npm run build
```

打包後的文件位於 `out` 文件夾中，可部署到：
- Vercel
- Netlify
- GitHub Pages
- 任何支持靜態文件的服務器

## 注意事項

1. **圖片管理**: 所有產品圖片位於 `public/images` 文件夾，新增產品時請確保圖片已上傳
2. **數據存儲**: 後台管理系統使用 localStorage 存儲數據，清除瀏覽器數據會導致數據丢失
3. **SEO優化**: 每個頁面都有獨立的 SEO 元標籤，可在後台設定頁面修改
4. **響應式設計**: 全站支持電腦、平板、手機三種設備，無跑版錯位
5. **性能優化**: 圖片懒加載、代碼精簡、無多餘外掛，海外訪問高速載入
